## 二叉树
树是数据结构，由节点构成，本身具有val属性
单链表每个结构只有一个指针 next
树具有两个指针 left right

- 根节点
- 叶子节点
- 边
- 节点 left right
- 满二叉树
- 完全二叉树

给定一个二叉树，返回其按层次遍历的节点值。 （即逐层地，从左到右访问所有节点）。
例如:
给定二叉树: [3,9,20,null,null,15,7]：
    3
   / \
  9  20
    /  \
   15   7
返回其层次遍历结果：
[
  [3],
  [9,20],
  [15,7]
]
遍历
  一次一层 队列
