一个草原上有500只老虎1只羊
虎只能吃一整只羊，然后会变成羊

1. 不吃，吃了会变成羊
    只有一只虎时，吃
2. 吃
    有两只老虎一只羊，不吃
3. 不吃，做朋友
    有三只老虎一只羊时，吃
    四只老虎，不吃
    五只老虎，吃
    ……
    五百只老虎，不吃

    大事情难解决，分成小事情
    大事化小，递归
    f(n) = !f(n-1)
    边界 n-1 吃，n=2 不吃

动态规划(Dynamic Programing)是常考算法，上楼梯问题
    三个术语：
        最优子结构 500-> 499-> ... 2->1
        边界 n=1 吃，n=2 不吃
        状态转移公式 虎变羊 f(n) = !f(n-1)

    楼梯台阶有12级，一步只能走一阶或两阶，请问有多少走法？
        最优子结构 f(12)
            f(12) f(11) f(10)
        边界 1级 1 两级 2
        状态转移公式 f(12) = f(11) + f(10)
